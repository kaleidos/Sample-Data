h2. Sample Data

Offers a service to create several types of sample data, like names, emails, numbers, etc 

h3. Installation

Execute the following from your application directory (pending approval):

{code}
$ grails install-plugin sample-data
{code}

h3. Usage

{code}
def sampleDataService // instanciate the service

sampleDataService.email() => "veritatis@fugiat.com"

{code}
